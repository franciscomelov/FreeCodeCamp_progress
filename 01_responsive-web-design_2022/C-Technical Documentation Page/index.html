¿Que es programacion?

Programar es crear una serie de instrucciones, en un lenguaje que las computadoras puedan entender, estas instrucciones detallan tareas especificas que seran ejecutadas, tal y como las escribiste. Por lo que hay que escribir de manera detallada y ser muy especifico. Las computadoras haran exactamente lo que les digamos, a menos que haya algun problema con ellas. A la hora de escribir un programa es importante tener en cuenta:

  • Pensar de forma lógica
  • Crear soluciones eficientes
  • Cada paso debeser especifico y detallado

Todo el Mundo Puede Programar

Para empezar no es nesario, ser un genio de las computadoras o ser muy buena para las matematicas. Solo es necesario algo de curiosidad. y buscar uno de los tantos tutoriales que hay en internet comenzar a seguir las instrucciones, aprender las reglas y las sintaxis del lenguaje. No importa la edad, tu experiencia previa o que te dediques. El aprender una nueva habilidad siempre te beneficiara en algo Por ejemplo, el siguiente codigo sumara dos numeros y mostrara el resultado en una ventana de alerta;

let a = 10;
let b = 20;
let resultado = a + b;
alert(resultado)

Talvez la primeravez que lo veas puede ser algo confuso, pero una vez aprendas la sintaxis básica, en este caso de Javascript, con solo ver el codigo sabras que hace.
